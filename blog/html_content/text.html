Etiquetas de Texto en HTML

El texto en HTML es el que se incluye dentro de otros elementos HTML para estructurar y presentar contenido en una página web.


  1. Elementos de Texto en Bloque:


    • <p> Define un párrafo. Se utiliza para agrupar bloques de texto
      • <p>Este es un párrafo de ejemplo.<p>

    • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Define encabezados, siendo <h1> el más importante y <h6> el menos importante.
      • <h1>Encabezado de nivel 1</h1>
      <h2>Encabezado de nivel 2</h2>

    • <div>: Define una división o sección genérica en un documento. Se usa para agrupar elementos en bloque.
      • <div>
      <p>Texto dentro de un div.</p>
      </div>

  2. Elementos de Texto en Línea:


    • <span>: Define una sección en línea dentro de un documento. Se usa para agrupar texto en línea.
      • <p>Este es un <span>texto en línea</span> dentro de un párrafo.<p>

    • <a>: Define un hipervínculo.
      • <a href="https://www.example.com">Visita Example</a>

    • <strong>: Define texto fuerte, generalmente mostrado en negrita.
      • <p>Este es un texto <strong>importante</strong>.</p>

    • <em>: Define texto enfatizado, generalmente mostrado en cursiva.
      • <p>Este es un texto <em>enfatizado</em>.</p>


Etiquetas para texto semántico
Etiqueta Ejemplo
